# YT-Channel-Stats-for-Data-Science-using-API

This code uses YouTube's API to grab the latest 50 videos from any channel, to give you a Pandas DataFrame containing:
1. `video_id`
2. `video_title`
3. `upload_date`
4. `view_count`
5. `like_count`
6. `comment_count`
for the latest 50 videos.
